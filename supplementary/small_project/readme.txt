# Конфигурирование проекта с cmake

cmake ../ 

первый и единственный аргумент это путь к CMakeLists.txt


можно сконфигурировать cmake на работу с другими средствами сборки (они тоже должны быть установленны), например Ninja 
cmake ../ -G ninja



при конфигурировании проекта можно выбрать конкретный компилятор, которым хотите компилировать ваши программы 
если он у вас уже установлен 

cmake ../   -DCMAKE_CXX_COMPILER=clang++


можно указать тип сборки -DCMAKE_BUILD_TYPE=Debug или -DCMAKE_BUILD_TYPE=Release 
// Debug - используется для отладки программ во время разработки, программа собранная как Debug работает медленней 
// но позволяет пользоваться отладчиком  
// Release - название говорит само за себя 

cmake ../  -G ninja -DCMAKE_BUILD_TYPE=Debug


cmake --buid .
